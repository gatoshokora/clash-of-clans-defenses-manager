# Clash of Clans Defense Manager

## 概要

概要

Clash of Clans（クラクラ）の防衛戦績を記録・管理するためのデスクトップアプリです。主にClash of Clansをプレイしているユーザー向けに作成しました。リーグごとの防衛データを登録・管理できるほか、集計結果やグラフを表示して防衛傾向を分析できます。

## 主な機能

* 防衛データの登録
* 防衛データの編集・削除
* CSVファイルへの自動保存・起動時の自動読み込み
* リーグ・リーグ期間ごとの一覧表示
* リーグ・リーグ期間ごとの詳細画面表示
* 防衛回数・平均破壊率・平均★数・平均トロフィー数などの集計表示
* 防衛データ一覧の表示
* 編成割合・★割合の円グラフ表示
* 入力値のバリデーション（不正なデータの登録防止）

## 使用技術

- Python 
- Tkinter（GUI）
- Matplotlib（グラフ表示）
- csv（データ保存・読み込み）

## 工夫した点

- リーグ・リーグ期間ごとに防衛データを管理できるようにし、過去の防衛戦績を見返しやすい構成にしました。

- 集計結果だけでなく、編成割合や★割合を円グラフで表示し、防衛傾向を視覚的に把握できるようにしました。

- 集計結果の近くにグラフ表示ボタンを配置し、数値とグラフを続けて確認できるようにしました。また、詳細画面から編集・削除を行えるようにし、操作しやすい画面構成を意識しました。

- 詳細画面では表示データと元データの位置を対応付けるために detail_indexes を利用し、正しいデータを編集・削除できるようにしました。
